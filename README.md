# Marcello Gomes Farias | Resume Web

Pagina estatica pronta para publicacao via GitHub Pages.

## Publicacao no GitHub Pages

1. Envie todos os arquivos para um repositorio no GitHub.
2. Abra `Settings` -> `Pages`.
3. Em `Build and deployment`, escolha `Deploy from a branch`.
4. Selecione a branch principal e a pasta `/ (root)`.
5. Salve.

## Estrutura

- `index.html`: conteudo principal da pagina
- `styles.css`: layout, tipografia, responsividade e animacoes
- `script.js`: interacoes, tabs e efeitos de entrada
- `assets/images/`: imagens usadas na pagina
- `assets/docs/`: PDF do curriculo para download
